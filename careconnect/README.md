# CareConnect Project Directory

This directory contains the CareConnect healthcare management platform with separate frontend and backend applications.

## Quick Links

### 📖 Main Documentation
See the [Main README](../README.md) for:
- Project overview
- System architecture
- Feature list
- Getting started guide

### 🚀 Frontend Setup
Go to [Frontend Directory](./frontend) for:
- React + Vite development setup
- Installation instructions
- Development server startup
- Build commands

**Quick Start:**
```bash
cd frontend
npm install
npm run dev
```

### 🔧 Backend Setup
Go to [Backend Directory](./backend) for:
- Node.js + Express server setup
- MongoDB configuration
- Environment variables
- API documentation

**Quick Start:**
```bash
cd backend
npm install
npm run dev
```

## Directory Structure
